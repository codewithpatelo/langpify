Installation
============

You can install Langpify using pip:

.. code-block:: bash

    pip install langpify

Or using Poetry:

.. code-block:: bash

    poetry add langpify

Development Installation
-----------------------

For development, you can clone the repository and install it in development mode:

.. code-block:: bash

    git clone https://github.com/yourusername/langpify.git
    cd langpify
    poetry install
